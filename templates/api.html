交通可视化系统 概述 人口迁徙情况 区域流量趋势 区域流量密度 区域流量预测 数据接口

数据接口概述

1. 添加数据接口

HTTP Method POST

HTTP Path /adddata

请求参数

参数名称 参数位置 参数类型 是否必须 作用
tra_data path map 是 交通流量数据

请求示例

http://47.95.147.13:9999/adddata/

响应参数

参数名称 参数位置 参数类型 是否必须 作用
code req body int 是 状态码
des req body String 是 状态描述

响应示例

{
              "code": 100,
              "des": "Added successfully",
            }

2. 人口迁移情况接口

HTTP Method POST

HTTP Path /getflydata

请求参数

参数名称 参数位置 参数类型 是否必须 作用
region req body String 是 需要查询的交通区域名称

请求示例

http://47.95.147.13:9999/getflydata/

响应参数

参数名称 参数位置 参数类型 是否必须 作用
code req body int 是 状态码
des req body String 是 状态描述
startpoints req body list 是 近30秒内的交通流量起点坐标集合
endpoints req body list 是 近30秒内的交通流量终点坐标集合

响应示例

{
              "code": 100,
              "des": "success",
              "startpoints":[
                  [-73.98966979980467, 40.75914764404297], 
                  [-73.98416137695312, 40.759010314941406],
                  ......
              ],
              "endpoints":[
                  [-73.95948028564453, 40.77957534790039], 
                  [-73.96556091308594, 40.7601318359375],
                  ......
              ],
              
            }

3. 区域流量密度接口

HTTP Method POST

HTTP Path /getheatdata

请求参数

参数名称 参数位置 参数类型 是否必须 作用
startdate req body String 是 查询开始时间
enddate req body String 是 查询结束时间

请求示例

http://47.95.147.13:9999/getheatdata/

响应参数

参数名称 参数位置 参数类型 是否必须 作用
code req body int 是 状态码
des req body String 是 状态描述
mapdata req body list 是 结果

响应示例

{
              "code": 100,
              "des": "success",
              "mapdata":[
                  {
                      "LocationID":201,
                      "weight":10,
                  },
                  {
                      "LocationID":117,
                      "weight":3,
                  },
                  ...
              ]
              
            }

5. 区域流量预测接口

HTTP Method POST

HTTP Path /getpredata

请求参数

参数名称 参数位置 参数类型 是否必须 作用
region req body String 是 需要查询的交通区域名称
startdate req body String 是 查询开始时间
enddate req body String 是 查询结束时间
granularity req body String 是 预测粒度

请求示例

http://47.95.147.13:9999/getpredata/

响应参数

参数名称 参数位置 参数类型 是否必须 作用
code req body int 是 状态码
des req body String 是 状态描述
tradata req body map 是 给定时间内的交通流量统计
predata req body map 是 预测结果数据

响应示例

{
              "code": 100,
              "des": "success",
              "tradata":{
